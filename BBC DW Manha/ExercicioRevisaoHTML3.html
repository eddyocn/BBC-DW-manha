Exemplo 3 HTML - Estrutura de um documento HTML

Este exemplo mostra a Estrutura minim de um documento HTML. A seguirsão descritas as principais tags utilizadas.

  • <!DOCTYPE html > Define o tipo do documento html
  • <html >É o elemento central, o container, de todo documento html.
  • <head >contém os metadados que serão interpretados para arquivos externos e meta arquivos de configuração do documento.
  • <body >Usamos essa tag para deixar visivel todo conteúdo de pagina ou documento HTML

;Abaixo está o codigo completo deste exemplo

            <!DOCTYPE html>
        <html lang="pt-BR">
          <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Exemplo 1 - Estrutura Básica</title>
          </head>
          <body>
            <h1>Bem-vindos à minha página!</h1>
            <p>Conteúdo principal aqui</p>
            <p>Rodapé da página</p>
          </body>
        </html>
