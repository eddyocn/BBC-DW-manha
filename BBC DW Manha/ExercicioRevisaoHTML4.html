Exemplo 4 HTML - Elementos HTML (Listas Ordenadas e Não Ordenadas)

Listas

Listas em html são representadas pelas tags <ol>(Lista Ordenadas) e <ul>(Lista Não Ordenada). essas listas possuem diversas finalidades, podendo ser utilizadas para exibir informações ou simplesmente serem usadas em menu

Toda vez que usarmos listas ordenadas ou não ordenadas deveremos utilizar a tag item <li>

Exemplo Lista Ordenada <ol>

  1. item de lista 1
  2. item de lista 2
    1. Subitem 2.1
    2. Subitem 2.2
    3. Subitem 2.3
    4. Subitem 2.4
  3. item de lista 3
  4. item de lista 4
  5. item de lista 5
    1. Subitem 5.1
    2. Subitem 5.2
    3. Subitem 5.3
      1. Subitem 5.3.1
      2. Subitem 5.3.2
      3. Subitem 5.3.3
      4. Subitem 5.3.4
    4. Subitem 5.4
  6. item de lista 6

Exemplo Lista Não Ordenada <ul>

  • item de lista 5

    • Subitem 5.2
    • Subitem 5.3
      • Subitem 5.3.1
      • Subitem 5.3.2
      • Subitem 5.3.3
      • Subitem 5.3.4
